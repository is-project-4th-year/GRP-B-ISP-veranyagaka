SRCNN Project

Enhancing Medical Imaging with Super-Resolution Convolutional Neural Networks

About the Project

This project implements a Super-Resolution Convolutional Neural Network (SRCNN) to improve the quality of low-resolution images by reconstructing fine details. It is inspired by the pioneering work of Dong et al. and adapted for medical imaging research.

Key Features

✔️ SRCNN model implementation
✔️ Training & evaluation scripts
✔️ Metrics: PSNR, SSIM
✔️ Image super-resolution inference
